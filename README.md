# Rasa_Chatbot
End to end rasa chatbot that has been trained and deployed to the cloud.
The endpoint is: https://dockersrc-sanchitsharma10.cloud.okteto.net/webhooks/rest/webhook you can use a servic like Postman to post JSON requests to it such as:
{
    "sender": "user",
    "message": "Hello"
}

For Headers just add "Content-Type" and its Value is "application/json" 

More details will be processed here shortly!
